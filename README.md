# Resume Optimization Agent

This is a resume optimization workflow built using n8n.

### Features:
- Parses job descriptions
- Extracts key skills
- Rewrites resume bullet points dynamically
- Outputs optimized resume file (txt)

### Tech Stack:
- n8n
- LLM API (gpt 4.0 mini)
- File system automation

### How to Use:
1. Import the JSON file into n8n
2. Add your LLM API key
3. Provide resume + job description
4. Run workflow
